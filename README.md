# 📊 Property Investment Analysis – Parramatta (NSW)

## 📌 Overview

This project presents a data-driven analysis of residential property prices in **Parramatta LGA**, with the objective of evaluating **investment opportunities** across three property categories:

* All dwellings
* Strata
* Non-strata

While Parramatta is the primary focus, additional data from **nearby LGAs (Middle Ring)** and **Inner Sydney** is incorporated to provide **contextual benchmarking and comparative insights**.

The analysis covers historical data from **1993 to 2017**, with additional **2021 data** used to validate the robustness of earlier insights and recommendations.

---

## 🎯 Key Contributions

* Cleaned and structured multi-source housing datasets
* Adjusted property prices for inflation using **CPI**
* Conducted **comparative analysis** between Parramatta, surrounding Middle Ring areas, and Inner Sydney
* Built clear visualisations to communicate trends and relative performance
* Generated actionable insights on investment advantages and risks
* Developed a data-backed investment recommendation
* Validated findings using out-of-sample data (2021)

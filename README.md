# 📊 Property Investment Analysis – Parramatta (NSW)

## 📌 Overview

This project presents a data-driven analysis of residential property prices across New South Wales, with a focus on **Parramatta LGA**. The objective is to evaluate **investment opportunities** across three property categories:

* All dwellings
* Strata
* Non-strata

The analysis covers historical data from **1993 to 2017**, with additional **2021 data** used to validate the robustness of earlier insights and recommendations.

---

## 🎯 Key Contributions

* Cleaned and structured raw datasets for analysis
* Adjusted property prices for inflation using **CPI** to ensure comparability over time
* Built clear visualisations to communicate trends and patterns
* Generated actionable insights on investment advantages and risks
* Developed a data-backed investment recommendation
* Validated findings using out-of-sample data (2021)
